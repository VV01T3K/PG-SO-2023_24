Main Convedit window:
    - Add {slect_file|select_files} - dodanie plików na listę
        --> You can add multiple copies of the same file
    - Remove {multi_select|single_select} - usunięcie plików z listy
    - Combine {multi_select} - otwarcie okna do łączenia plików
    - Edit {single_select} - otwarcie okna do edycji pliku
    - Play {single_select} - odtworzenie pliku
    - About - informacje o programie

File Selection window:
    --> aka default system file explorer
    - Type_Filter { media, wideo, audio } - filtracja plików po typie
    - Select {multi_select|single_select} - wybór pliku
    - Ok - zatwierdzenie wyboru
    - Cancel - anulowanie wyboru

Edit window:
    --> You can edit only one file at a time
    --> Supported file formats: mp4, avi, mkv, webm, mp3, wav, flac, ogg
    --> Some files may not work correctly,
        try changing the output format or make fewer changes at once
    - Save - zapisanie jako nowy plik lub nadpisanie
    - Cancel - anulowanie zmian

Combine window:
    --> Multiple audios and videos can be combined into one file
    --> You can combine one audio file with one video file
        But you can't combine mutliple audio files with multiple video files
    --> The order of the files can be changed by dragging them on the list
        - Merge (video+audio) - nałożenie ne siebie plików
        - Merge (audio+audio) - nałożenie ne siebie plików w kolejności listy
        - Concat (video+video) - połączenie plików w kolejności listy
        - Cancel - anulowanie połączenia
